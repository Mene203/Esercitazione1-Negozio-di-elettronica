# ESERCITAZIONE 07 - Negozio di Elettronica
## 📌 Descrizione del progetto
Il progetto simula il lavoro di due sviluppatori che collaborano alla gestione di un inventario di un negozio di elettronica utilizzando Git e GitFlow.

Il sistema prevede due versioni del progetto:
- **Versione 1**, rappresenta la struttura iniziale semplice dell’inventario
- **Versione 2**, rappresenta l’evoluzione del progetto con una riorganizzazione delle cartelle e l’aggiunta di nuove categorie di prodotti

Lo scopo è simulare il lavoro collaborativo tra due sviluppatori, con sviluppo parallelo, gestione delle modifiche e organizzazione del codice tramite versionamento.
